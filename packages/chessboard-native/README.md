# @vibechess/chessboard-native

A controlled, rules-free React Native chessboard component.

This package is an unpublished Phase 0 shell. It exports the controlled
position, move, annotation, renderer, accessibility, and typed-error contracts,
plus a disabled visual frame used to verify package builds and Expo workspace
consumption.

The frame does not consume the new contracts yet. Position normalization,
static rendering, and interaction land in later phases; exported data types are
not a claim that those runtime behaviors are implemented.

Do not use this package in production yet.
